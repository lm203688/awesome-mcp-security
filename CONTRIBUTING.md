# Contributing to Awesome MCP Security

Thank you for your interest in contributing!

## How to Contribute

### Add a Security Tool

1. Fork this repository
2. Add the tool to the appropriate section in README.md
3. Include: tool name, GitHub link, brief description
4. Submit a Pull Request

### Report a Vulnerability

1. Add it to the "Vulnerability Database" section
2. Include: ID, title, severity, description
3. If it's a real-world incident, add to "Real-World Incidents"

### Add Best Practices or Research

1. Add to the appropriate section
2. Include: title, author/source, link
3. Ensure content is relevant to MCP security

## Guidelines

- Keep descriptions concise (1-2 sentences)
- Use tables for structured data
- Verify all links are working
- Sort tools by stars (descending)

## Code of Conduct

Be respectful. Focus on improving MCP security for everyone.
